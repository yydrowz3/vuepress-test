# 自动驾驶课程分享

Coursera上有一门自动驾驶的专项课程，由多伦多大学开设，个人很推荐。

https://www.coursera.org/specializations/self-driving-cars (opens new window)

这个系列的课程分为四个部分，分别是：

  1. Introduction to Self-Driving Cars
  2. State Estimation and Localization for Self-Driving Cars
  3. Visual Perception for Self-Driving Cars
  4. Motion Planning for Self-Driving Cars

如果不是想获得Coursera的证书，可以选择旁听，不需要缴费。根据这里边的介绍可以你自己应该能知道自动驾驶会涉及哪些方面的知识。每个章节有对应的PPT可以下载，还有作业可以练手，final project也有做过的人把代码放在github上，论坛里也有别人的讨论。（缺点是看视频好像要翻墙？反正我不靠校园网翻墙的话视频似乎加载不出来）
